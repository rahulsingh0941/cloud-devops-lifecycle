About This Application

This cloud-native web application demonstrates the complete DevOps lifecycle using modern tools and practices. It's built for scalability, automation, and continuous delivery.

Technologies Used

  • Ansible – Configuration managemnt tool
  • Terraform – Infrastructure as Code (IaC)
  • Jenkins – CI/CD Automation
  • Docker – Containerization
  • Kubernetes – Container Orchestration
  • AWS – Cloud Hosting

Purpose

This project showcases a scalable DevOps pipeline for deploying production-ready web apps in the cloud. It integrates automation, monitoring, and modern development workflows to ensure efficiency and reliability.
